Cyber Incident Response Case Manager

Local-first educational prototype for organizing internal cyber incidents, artifacts, indicators, timelines, response actions, ATT&CK mappings, and incident reports.

Educational prototype: This is an educational prototype using browser localStorage and fictional sample data. It is not production-ready and should not be used for real sensitive incident data without authentication, encrypted storage, backend-enforced access controls, audit logging, secure backend integrations, backups, and security/legal review.
